# 从素材到爆款：卡兹克风格写作流程

> 有素材只是起点，组织成情感曲线才是爆款的关键

---

## 前置条件：素材清单

开始写作前，确认你有：

- [ ] **人物档案**：姓名、年龄、背景、现状
- [ ] **核心冲突**：TA遇到了什么困难/挑战？
- [ ] **转折点**：什么改变了TA的轨迹？
- [ ] **具体成果**：量化结果（数字、排名、收入）
- [ ] **一手素材**：访谈记录、截图、聊天记录、文档
- [ ] **独家发现**：与常识相反的洞察
- [ ] **金句/原话**：人物的原话或关键对话
- [ ] **你的连接**：你和这个故事的关联/共鸣

---

## 写作流程（7步成文）

### Step 1: 素材拆解（10分钟）

把素材按时间线整理：

```
时间线模板：
- 起点：[人物背景，困境]
- 触发：[什么事件让TA开始改变]
- 过程：[关键行动，遇到的困难]
- 转折：[意外发现/关键决策]
- 结果：[具体成果]
- 启示：[给你的冲击/思考]
```

**示例**（陈雨欣）：
```
- 起点：二本，网络与新媒体专业，喜欢北大但觉得遥远
- 触发：大一时接触到社会学，被吸引
- 过程：自学4个月社会学发现太难，转考社会工作；无专业、无学长、无支持
- 转折：去年5月遇到Chatbox/DeepSeek；发现"不用AI"比"用AI"更重要
- 结果：考上北大社工，全国只招11人
- 启示：经济差比信息差更残酷；工具免费但能力是自己的
```

---

### Step 2: 情感曲线设计（10分钟）

卡兹克文章的情感曲线：

```
好奇 → 真实感 → 同情 → 希望 → 实用价值 → 惊喜 → 共鸣 → 记忆点
```

**对应段落**：

| 情感 | 段落 | 功能 |
|------|------|------|
| 好奇 | 钩子（1-2段） | 制造悬念，让人想读下去 |
| 真实感 | 发现故事（2-3段） | "我是怎么知道这个故事的" |
| 同情 | 困难堆叠（3-4段） | 让读者觉得"这太难了" |
| 希望 | 转机出现（2段） | 转折点，新的可能性 |
| 实用价值 | 方法细节（4-5段） | "她是怎么做到的" |
| 惊喜 | 核心洞察（2-3段） | 反转，与常识相反 |
| 共鸣 | 升华思考（3-4段） | 从个人到社会议题 |
| 记忆点 | 金句结尾（1-2段） | 让读者记住并转发 |

---

### Step 3: 钩子设计（关键）

**公式**：反差 + 具体 + 省略号

```
❌ 普通：一个女生用AI考上了北大
✅ 卡兹克：一个二本的女生，用免费的AI考上了北大

关键升级：
- 二本 vs 北大（反差）
- 免费的（具体细节，暗示经济困难）
- 省略号（让人想知道"怎么做到的"）
```

**其他钩子示例**：
- "22岁那年，我开出了一张5万美元的支票。不是投给知名基金，是投给MIT一个默默做项目的同学。"
- "我们公司有个运营，他母校的校友群炸了。一个二本学校，出了个考上北大的。"

---

### Step 4: 渐进式揭秘（核心技巧）

不要一次性给完信息，要层层剥开：

```
❌ 错误：陈雨欣，21岁，安徽人，二本，网络与新媒体专业，喜欢北大，高考失利...

✅ 正确：
- 第1层：她叫陈雨欣，21岁，安徽人（基础信息）
- 第2层：她从高中就喜欢北大，但高考失利去了二本（冲突）
- 第3层：学的是网络与新媒体，学校没社工专业（困难+1）
- 第4层：北大社工全国只招11人，北京压分狠（困难+2）
- 第5层：身边没学长学姐，父母不了解，同学不知道她在考研（困难+3）
```

**原则**：每段只揭示一层，让读者自己拼凑出"这太难了"的感受。

---

### Step 5: "我"的叙事（建立信任）

卡兹克文章中"我"的作用：

1. **引入故事**："我们公司有个运营小伙伴..."
2. **表达兴趣**："我其实也非常非常感兴趣..."
3. **推动行动**："我拜托小伙伴去联系..."
4. **情感共鸣**："我到现在还记得，我上大学的时候..."
5. **引导发现**："但真正让我觉得厉害的地方是..."
6. **价值观输出**："这让我意识到..."

**写作技巧**：
- 每300-500字出现一次"我"
- "我"的情感要真实（好奇→惊讶→感动→思考）
- 用"我"的经历建立与读者的连接

---

### Step 6: 反转设计（爆款关键）

**公式**：铺垫 → 预期 → 打破预期

```
铺垫：
"现在大多数人用AI，上来就是让它帮忙读完这本书，整理笔记，输出总结。恨不得让AI把所有事都干了。"

预期：
"但陈雨欣她不会，她只会当读书读到卡点的时候用AI。"
（读者预期：她用得少，更自律）

打破预期：
"但真正让我觉得这个女生厉害的地方，其实还不是她怎么用AI，因为这些我觉得大家肯定都听过了，最有意思的是。她怎么不用AI。"
（反转：重点不是"用"，是"不用"）
```

**反转后的展开**：
- 解释"不用"的具体做法（闭卷模拟）
- 解释"为什么不用"（怕AI思路替代自己）
- 升华：什么该给AI，什么必须自己来

---

### Step 7: 金句植入（记忆点）

**金句来源**：
1. **人物原话**："你是自己人生的神。我只是那个，在你感觉完了的时候，轻轻说了一句再走一步试试的人。"（DeepSeek回复）
2. **名言引用**：荣格："I am not what happened to me, I am what I choose to become."
3. **你的总结**："工具是免费的，但，能力是自己的。"

**金句位置**：
- 文章中部：1个（反转点）
- 文章后部：1-2个（升华点）
- 结尾：1个（收尾，点题）

## 完整结构模板

```
【钩子】1-2段
反差 + 具体 + 省略号

【发现故事】2-3段
"我是怎么知道这个故事的"
"为什么我想写下来"

【人物介绍】2-3段
渐进式揭秘：基础信息 → 背景 → 困难1 → 困难2 → 困难3

【转机出现】2段
什么改变了TA的轨迹
新的可能性

【方法细节】4-5段
TA是怎么做到的
具体做法 + 数据 + 截图引用

【核心洞察】2-3段
反转：与常识相反的发现
"真正厉害的是..."

【升华思考】3-4段
从个人到社会议题
你的价值观输出

【金句结尾】1-2段
人物寄语/名言
行动号召（点赞转发星标）
```

## 写作检查清单

完成初稿后检查：

- [ ] 首句有反差（二本 vs 北大）
- [ ] 前3段内出现"我"
- [ ] 困难是渐进堆叠的，不是一次性给完
- [ ] 有具体数字（30万字、1000元生活费、10元/顿）
- [ ] 有真实截图/对话引用
- [ ] 有1个反转（预期→打破预期）
- [ ] 有3个金句（中部、后部、结尾）
- [ ] "我"出现10次以上
- [ ] 情感曲线：好奇→同情→希望→惊喜→共鸣
- [ ] 结尾有行动号召

## 示例：从素材到文章

### 素材
```
人物：张三，外卖小哥，25岁
冲突：用AI自学编程，3个月拿到大厂offer
困难：每天送餐12小时，只能睡5小时，没电脑用手机学
转折：发现"教AI教自己"比"让AI写"更有效
成果：字节跳动，年薪35万
洞察：不是AI帮他成功，是AI让他有时间成功
```

### 钩子
```
一个外卖小哥，在等餐的间隙，用AI考上了字节跳动。

年薪35万。

他每天只睡5个小时。
```

### 渐进式揭秘
```
他叫张三，25岁，湖南人。

高中没毕业就出来打工，送过快递，端过盘子，最后在美团当骑手。

每天工作12小时，早上6点起床，晚上10点收工。

他没有电脑。所有的学习，都是在等餐的间隙，在电动车上，用一部用了3年的小米手机完成的。

（困难堆叠：没学历、没时间、没设备）
```

### 反转
```
现在大多数人用AI，都是直接说"帮我写个爬虫"。

但张三不会。

他会让AI先解释什么是爬虫，然后一步一步教他写，每一行代码都要自己敲。

真正让我觉得厉害的地方，不是他用AI学编程。

是他从来不让AI直接写代码。

他说："我怕我学不会。"
```

## 关键提醒

1. **素材是基础**：没有真实素材，技巧无用
2. **情感是核心**：让读者跟着你的情感走
3. **渐进是技巧**：不要一次性给完信息
4. **反转是爆点**：必须有与常识相反的发现
5. **金句是记忆**：让读者有转发的东西
