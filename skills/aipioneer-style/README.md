# AI Pioneer 公众号排版样式

三种排版风格，适配不同场景。

## 版本对比

| 版本 | 名称 | 主色 | 特点 | 适用场景 |
|------|------|------|------|----------|
| **v1.0** | 纯黑极简 | `#000000` | 标签 + 章节数字重叠 | 极简现代 |
| **v2.0** | 深海宝蓝 | `#0E61AC` | 品牌头图 + 奶润米杏引用 | 温暖专业 |
| **v3.0** | 章节数字优化 | `#000000` | 数字在标题上方（无遮挡） | 清晰阅读 |

## 快速使用

```bash
# v1.0 - 纯黑极简
npx aipioneer-publisher publish "链接" --theme ai-pioneer-v1

# v2.0 - 深海宝蓝
npx aipioneer-publisher publish "链接" --theme ai-pioneer-v2

# v3.0 - 章节数字优化
npx aipioneer-publisher publish "链接" --theme ai-pioneer-v3
```

## 详细说明

### v1.0 纯黑极简版

- **标签**: `AI Pioneer · 深度观点` 黑色圆角标签
- **强调**: `#000000` 纯黑粗体
- **引用**: `rgba(0,0,0,0.06)` 浅灰背景
- **章节**: 80px 数字与标题重叠（-48px margin）

### v2.0 深海宝蓝版

- **头图**: 品牌头图，0px 间距
- **主色**: `#0E61AC` 深海宝蓝
- **引用**: `#FAF2E0` 奶润米杏背景
- **章节**: 与 v1 相同，数字重叠

### v3.0 章节数字优化版

- **基于**: v1.0 纯黑配色
- **改进**: 章节数字独立一行，标题在下方
- **优势**: 无遮挡，更清晰
